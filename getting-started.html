Prerequisites

Java 11 or later

JSoagger requires Java version 11. Download and install JDK version 11 or later.


>  java -version 
   openjdk version "11.0.1" 2018-10-16
   OpenJDK Runtime Environment 18.9 (build 11.0.1+13)
   OpenJDK 64-Bit Server VM 18.9 (build 11.0.1+13, mixed mode)

Maven

Download and install latest version of maven


> mvn -version
  Apache Maven 3.5.2
  Default locale: fr_FR, platform encoding: UTF-8

Docker container

Download and install latest version of docker desktop for your OS.


> docker version

  Go version:        go1.12.5

  Built:             Thu Jul 25 21:18:17 2019

Create and build project

Create project

Create a new project with jsoagger maven archetype:


> mvn archetype:generate -DarchetypeGroupId=io.github.jsoagger -DarchetypeArtifactId=jsoagger-fullstack-archetype
 -DarchetypeVersion=LATEST -DgroupId=myprojectgroupid -DartifactId=myproject

Build


> cd myproject
> mvn clean install -P docker,h2

Run


> docker-compose -f compose-dev up
