Soagg manager

Spring shell 2.0 CLI

Soagg manager is an interactive console where user can interact with the backend server through textual commands.

Visit spring shell website for more information https://docs.spring.io/spring-shell/docs/2.0.0.RELEASE/reference/htmlsingle/

Running from docker compose

If you have started containers with docker compose, then connect to manager will following command: 


> docker ps												
> docker -it exec soagg-manager bash /runcli.sh

Running from docker

Pull the image from docker hub or build image locally and run it. Manager must be on same network as the server it want to manage. 

					
> docker pull jsoagger/soagg-manager:latest										
> docker run -d --rm --network=soagg_network --name soagg_manager 
-e MANAGED_BACKEND_REST_URL='http://soagg-foundation-srv:8080/jsoagger'
 -v $CONFIG_LOCATION:/spring-config/ jsoagger/soagg-manager:latest

Running from Java

You can download latest version of soagg-manager from maven central https://search.maven.org/search?q=jsoagger. 


> java -jar jsoagger-manager-sb.jar --remote.host.url=http://soagg-foundation-srv:8080/jsoagger

Basic commands

Ping managed server


[http://soagg-foundation-srv:8080/jsoagger] > ping
									

									
									

										
Login managed server

									

									



[http://soagg-foundation-srv:8080/jsoagger] > login epadmin epadmin
