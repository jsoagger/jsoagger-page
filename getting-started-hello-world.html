Create and build a project

Make sure that Maven, Java 11 and docker are installed before begining.

Create project

Create a new project with jsoagger maven archetype


> mvn archetype:generate -DarchetypeGroupId=io.github.jsoagger -DarchetypeArtifactId=jsoagger-fullstack-archetype
 -DarchetypeVersion=LATEST -DgroupId=myprojectgroupid -DartifactId=myproject

Build


> cd myproject
> mvn clean install -P docker,h2

Run it

Compose up!


> docker-compose -f compose-dev.yaml up

Browse web user interface

Open a web browser and type url : http://localhost:3000/jsoagger

Browse h2 database console

Open a web browser and type url : http://localhost:81/
