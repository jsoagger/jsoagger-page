Open source and free software

JSOAGGER is a framework for building cross platform Java applications. Its integrates technical framworks such as spring-boot, JPA, Hibernate, ReactJS, JavaFX11 and Docker container in order to build robust applications based on current market standards.

ReactJS web UI

ReactJS is a very popular javascript framework for building modern user interface. JSOAGGER integrates web ui and components built on top of ReactJS.

JavaFX Desktop UI

Build powerfull and highly customizable desktop application with modern JavaFX 11 and 12. JSOAGGER integrates JavaFX components, Http connector and desktop integration.

Mobile UI

Build powerfull and highly customizable mobile application with modern JavaFX 11 and 12. JSOAGGER integrates JavaFX components, Http connector and mobile integration, inculing IOS and Androïd.

Embedded UI

Build embedded application with modern JavaFX 11 and 12. JSOAGGER can integrate custom platforms like Raspbery PI or Arduino.

Command Line Interface

Manage backend with a console built with spring shell. You can manage your business, load and export datas, manage application state and more.

Capitalize and Reuse

TECHNICAL PLATFORM

JSOAGGER integrates famework and tools like Hibernate, Spring, Spring-boot, JPA, Docker container, ReacjtJS, JavaFX11 and more have been integrated in order to let developpers have more focus on business needs.

FUNCTIONAL PLATFORM

Functional bricks like authentication, authorization, version history and lifecycle management are ready to be specialized or extended to serve your business.

CROSS PLATFORM GUI

HMIs can be deployed under any types of client including Web, IOS, Androïd, desktop and Embedded platforms. JSOAGGER uses ReactJS for the web and JavaFX 11 for Desktop, mobile and Embedded platforms integration.

RESPONSIVE AND ADAPTATIVE

As each business has its own way of interacting with the system, the platform have been built in order to ease the construction of intelligent and optimized GUIs.

VERTICAL INTEGRATION

Modularity is the heart of JSOAGGER. Each visual component is integrated with business services thanks to our different connectors.

BUILT IN COMPONENTS

Composed by a rich, customizable, reusable and extensible component library.
