Open source and free software

JSOAGGER is a framework for building fullstack and mutliplatform Java applications. It integrates technical frameworks like spring-boot, JPA, Hibernate, ReactJS, JavaFX11 and Docker container, with business modules like lifecycle, product structure and document management.
